# Dilukshan Niranjan — Portfolio

A responsive personal portfolio showcasing selected projects, technical expertise, and certifications.

Built with Astro, TypeScript, Lenis smooth scrolling, and custom CSS animations.

## Run locally

```sh
npm install
npx astro dev --background
```

The site will be available at `http://localhost:4321`.

Use `npx astro dev stop` to stop the background development server.

## Project structure

- `src/pages` — Site pages and project routes
- `src/components` — Shared Astro components
- `src/data` — Project and certification content
- `public` — Images, icons, and static assets
